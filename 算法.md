### 查找算法

- 线性查找

  ``` php
  $arr=[1,3,2,10,4,5];
  $beFind=10;
  
  foreach ($arr as $item){
    if ($item==$beFind){
      return $item;
    }
  ```

- 二分查找

  ```php
  function binary(array $arr, int $val, int $left, int $right)
      {
          if ($left > $right) {
              return -1;
          }
          $mid = ($left + $right) / 2;
  
          if ($val < $arr[$mid]) {
              return $this->binary($arr, $val, $left, $mid - 1);
          } elseif ($val > $arr[$mid]) {
              return $this->binary($arr, $val, $mid + 1, $right);
          }
  
          return $mid;
      }
  ```
